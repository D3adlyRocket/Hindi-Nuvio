# Hindi-Nuvio
Hindi compiled plugins for Nuvio.

## Installation

1. Open **Nuvio** app
2. Go to **Settings в†’ Local Scrapers**
3. Add this repository URL:
   ```
   https://raw.githubusercontent.com/D3adlyRocket/Hindi-Nuvio/refs/heads/main/manifest.json
   ```
4. Enable the scrapers you want to use
